leoMouse

leoMouse鼠标拖拽基础组件

引用：leoUi-mouse.js

依赖：jQuery, leoUi-tools.js

leoMouse鼠标拖拽基础组件

$('.leoUi').leoMouse( options )

使用说明：

examples：

                            $('.leoUi').leoMouse({

                                cancel:'input,textarea,button,select,option',//防止在指定元素上开始拖动

					            delay:0,//时间（以毫秒为单位），当鼠标按下后延迟多少秒激活拖动

					            distance:0,//当鼠标按下后移动多少px激活拖动

					            isSetCapture:false,//是否使用setCapture

					            mouseDownSelector:false//是否使用事件代理

                            })

options

事件：

_mouseCapture(/* event */)：

确定一个互动是否基于事件目标元素。返回 true；可以继续拖拽

arguments：

event:事件对象

returns(boolean)：不返回true，将停止之后的事件

写组件人判断是否有返回值

examples：

                            _mouseCapture: function(/* event */) { return true; }//内部调用

_mouseStart(/* oldEvent, event */)：

准备开始拖拽

arguments：

oldEvent:旧的事件对象（delay，distance > 0）

event:事件对象

returns(boolean)：返回 false，将停止拖拽

写组件人判断是否有返回值

examples：

                            _mouseStart: function(/* event */) {};//内部调用

_mouseDrag(/* event */)：

正在拖拽

arguments：

event:事件对象

returns：无

examples：

                            _mouseDrag: function(/* event */) {};//内部调用

_mouseStop(/* event */)：

结束拖拽拖拽

arguments：

event:事件对象

returns：无

examples：

                            _mouseStop: function(/* event */) {};//内部调用
