leoTooltip

leoTooltip提示框组件

引用：leoUi-tooltip.js

依赖：jQuery, leoUi-tools.js, leoUi-position.js, leoUi-effects.js

leoTooltip提示框组件

$.leoTooltip( options )

使用说明：

options

事件：

destroy()：销毁组件

arguments：无

returns：无

examples：

                            $.leoDialog( "destroy" );
                            $.leoDialog.destroy();

option(arguments)：设置或者获取组件选项的值

arguments：参考leoPlugIn中的option方法

returns：参考leoPlugIn中的option方法

examples：

                            $.leoDialog( "option" "a.b.c", 500 );
                            $.leoDialog.option( "a.b.c", 500 );

setDialogTitle(text)：设置都dialog标题

arguments：

text(string): 标题文本

returns：无

examples：

                            $.leoDialog( "setDialogTitle" "标题" );
                            $.leoDialog.setDialogTitle( "标题" );

clearDialogTimeout(id)：清除dialog和遮罩层的延时定时器

arguments：

id(string): 可以有"all", "modalShow", "modalHide", "dialogShow", "dialogHide",后面四个可以写一起写，以空格或者逗号隔开（比如：modalShow,dialogShow）

returns：无

examples：

                            $.leoDialog( "clearDialogTimeout" "modalShow,dialogShow" );
                            $.leoDialog.clearDialogTimeout( "all" );

dialogShow()：显示dialog和遮罩层

arguments：无

returns：无

examples：

                            $.leoDialog( "dialogShow" );
                            $.leoDialog.dialogShow();

modalDialogHide()：关闭dialog和遮罩层

arguments：无

returns：无

examples：

                            $.leoDialog( "modalDialogHide" );
                            $.leoDialog.modalDialogHide();

moveToTop()：移动当前的的dialog到最上面（isMoveToTop = true 才有效）

arguments：无

returns：无

examples：

                            $.leoDialog( "moveToTop" );
                            $.leoDialog.moveToTop();

modalState()：遮罩层的状态

arguments：无

returns：返回'open','close','opening','closeing'4中状态

examples：

                            $.leoDialog( "modalState" );
                            $.leoDialog.modalState();

dialogState()：dialog的状态

arguments：无

returns：返回'open','close','opening','closeing'4中状态

examples：

                            $.leoDialog( "dialogState" );
                            $.leoDialog.dialogState();

modalHide()：关闭遮罩层

arguments：无

returns：无

examples：

                            $.leoDialog( "modalHide" );
                            $.leoDialog.modalHide();

dialogHide()：关闭dialog

arguments：无

returns：无

examples：

                            $.leoDialog( "dialogHide" );
                            $.leoDialog.dialogHide();
