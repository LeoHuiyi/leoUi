leoTooltip

leoTooltip提示框组件

引用：leoUi-tooltip.js

依赖：jQuery, leoUi-tools.js, leoUi-position.js, leoUi-effects.js

leoTooltip提示框组件

$.leoTooltip( options )

使用说明：

options

事件：

destroy()：销毁组件

arguments：无

returns：无

examples：

                            $.leoTooltip( "destroy" );
                            $.leoTooltip.destroy();

option(arguments)：设置或者获取组件选项的值

arguments：参考leoPlugIn中的option方法

returns：参考leoPlugIn中的option方法

examples：

                            $.leoTooltip( "option" "a.b.c", 500 );
                            $.leoTooltip.option( "a.b.c", 500 );

tooltipPosition(notRandom)：设置都tooltip位置

arguments：

notRandom(boolean): 不要调用_randomPosition

returns：无

examples：

                            $.leoTooltip( "tooltipPosition" "true" );
                            $.leoTooltip.tooltipPosition( "false" );

clearDialogTimeout(id)：清除tooltip的延时定时器

arguments：

id(string): 可以有"show", "hide",可以写一起写，以空格或者逗号隔开（比如：show,hide）

returns：无

examples：

                            $.leoTooltip( "clearDialogTimeout" "show,hide" );
                            $.leoTooltip.clearDialogTimeout( "show" );

show()：显示tooltip

arguments：无

returns：无

examples：

                            $.leoDialog( "show" );
                            $.leoDialog.show();

hide()：关闭tooltip

arguments：无

returns：无

examples：

                            $.leoTooltip( "hide" );
                            $.leoTooltip.hide();
