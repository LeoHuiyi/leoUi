leoPlugIn

抽象化来创建有状态的 jQuery 插件

引用：leoUi-tools.js

依赖：jQuery

plugIn 创建jQuery插件

$.leoTools.plugIn( options, [methods] )：$.leoTools.plugIn自动创建统一的jQuery插件

options：

例子：

                    $.config({

                        level: 6,

                        nocache: false,

                        baseurl: static/js/,

                        alias: {

                            'leoUiCss': '../css/leoUi.css',//css别名

                            'leoUi-grid': 'leoUi/leoUi-grid',//js别名（可省略.js）
                        },

                        shim: {

                            jquery: {//别名

                                src: 'lib/jquery-1.11.1-min.js',//路径

                                exports: "$"//输出（可以是函数，return输出变量）

                            }

                        }

                    });

define 创建模块

define( [ name, deps ], factory )：define 是一个全局函数，通过该函数就可以创建一个符合 AMD 规范的模块，然后就可以通过leoUiLoad.require/$.require方法来加载这个模块。

arguments：

require 加载模块

leoUiLoad.require|$.require( list, [ factory ] )：require 方法是 leoUiLoad 提供的用于加载模块的方法，该模块只要遵循 AMD 的模块化规范，就可以加载。

arguments：

合并模块

leoUiLoad.js 提供了一个 leoUiLoadCombo 的合并工具，该合并工具运行在 node.js 的环境中，配置很简单。将下面的代码保存成 build.js：

                    require('./js/leoUiLoadCombo.js').leoUiLoadCombo({
                        baseUrl : '/js',//是需要合并的模块的基础路径。
                        uglifyUrl : '../uglify/uglify-js',//是压缩工具 uglify 的路径。
                        modules: [{//可以传一组待合并的数据。
                            input: ['src/i_class.js'],//一个或一组待合并的的模块，(多个模块压缩成一个文件就是一组)。
                            output: 'leo/js/src/i_class.js'//输出合并文件路径
                        }],
                        config:{//配置文件可以统一放在这里

                            level: 6,

                            alias: {

                                'leoUiCss': '../css/leoUi.css',

                                'public': 'src/public',

                            },

                            shim: {

                                jquery: {

                                    src: 'lib/jquery-1.11.1-min.js',

                                    exports: "$"

                                }

                            }

                        }
                    });

配置好后，可以直接在 cmd 中运行命令：node build，当然，前提是你必须安装了 node.js。
